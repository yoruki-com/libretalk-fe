LibreTalk is a React Native/Expo social app with authentication, chat, vibes, community, and profile features.

## Phase Details

### Phase 1: Auth Migration
**Goal**: Replace Auth0 with Logto in the frontend
**Depends on**: Nothing (backend already implemented)
**Success Criteria** (what must be TRUE):
  1. App authenticates users via Logto
  2. Token injection to API client works correctly
  3. Sign-in and sign-out flows function end-to-end
